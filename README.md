# ubuntu-must-know-code
THIS REPO CONTAINS THE MUST HAVE CODE FOR UBUNTU AND THE SHORTCUT MOST USED 
<pre>
1. Ctrl + Alt + T - Open Terminal

   Example: Ctrl + Alt + T



2. Ctrl + C - Copy

   Example: Select text and press Ctrl + C



3. Ctrl + X - Cut

   Example: Select text and press Ctrl + X



4. Ctrl + V - Paste

   Example: Place cursor and press Ctrl + V



5. Ctrl + Z - Undo

   Example: Press Ctrl + Z to undo the last action



6. Ctrl + Shift + Z - Redo

   Example: Press Ctrl + Shift + Z to redo



7. Ctrl + S - Save

   Example: Press Ctrl + S to save a document



8. Ctrl + N - New

   Example: Press Ctrl + N to create a new document



9. Ctrl + O - Open

   Example: Press Ctrl + O to open a file



10. Ctrl + Q - Quit

    Example: Press Ctrl + Q to quit an application



11. Alt + Tab - Switch between open applications

    Example: Press Alt + Tab to cycle through open applications



12. Ctrl + Alt + Arrow keys - Switch between workspaces

    Example: Press Ctrl + Alt + Right Arrow to move to the workspace on the right



13. Ctrl + Alt + Delete - Open System Monitor

    Example: Press Ctrl + Alt + Delete to open the System Monitor



14. Ctrl + Alt + L - Lock the screen

    Example: Press Ctrl + Alt + L to lock the screen



15. Alt + F2 - Run command

    Example: Press Alt + F2 to open the "Run a command" dialog



16. Ctrl + Alt + Del - Log out

    Example: Press Ctrl + Alt + Del to log out of the current session



17. Ctrl + Alt + Shift + Arrow keys - Move and resize windows

    Example: Press Ctrl + Alt + Shift + Right Arrow to move the window to the right



18. Ctrl + H - Show hidden files in file manager

    Example: Press Ctrl + H in the file manager to toggle hidden files visibility



19. Super key (Windows key) - Open Activities overview

    Example: Press the Super key to view the Activities overview



20. Ctrl + Alt + Shift + R - Start/stop screen recording

    Example: Press Ctrl + Alt + Shift + R to start or stop recording the screen

  THESE ARE THE TERMINAL CODES 

  this file includes all basic codes for ubuntu usage and imp ones which you regularly use 



1. ls - List files and directories in the current directory.

   Example: ls



2. cd - Change directory.

   Example: cd directory_name



3. pwd - Print working directory (displays the current directory path).

   Example: pwd



4. cp - Copy files or directories.

   Example: cp source destination



5. mv - Move or rename files or directories.

   Example: mv source destination



6. rm - Remove files or directories.

   Example: rm file_name



7. mkdir - Create a new directory.

   Example: mkdir directory_name



8. rmdir - Remove an empty directory.

   Example: rmdir directory_name



9. touch - Create an empty file or update file timestamp.

   Example: touch file_name



10. cat - Display the contents of a file.

    Example: cat file_name



11. nano/vi/vim - Text editors for creating or modifying files.

    Example: nano file_name



12. grep - Search for a specific pattern in files.

    Example: grep pattern file_name



13. chmod - Change file permissions.

    Example: chmod permissions file_name



14. chown - Change file owner.

    Example: chown owner:group file_name



15. ps - Display information about currently running processes.

    Example: ps



16. kill - Terminate a process.

    Example: kill process_id



17. df - Display information about disk space usage.

    Example: df



18. du - Display information about file and directory space usage.

    Example: du



19. top/htop - Display real-time system statistics.

    Example: top



20. scp - Securely copy files between systems.

    Example: scp source_file user@destination:/path
